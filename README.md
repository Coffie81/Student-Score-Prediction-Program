This project uses Python and linear regression to predict student academic performance from study hours. It demonstrates supervised learning and basic data analysis techniques.
# 📊 Student Score Prediction Using Study Hours

## 📌 Project Overview
This project builds a simple Machine Learning model that predicts a student's expected exam score based on the number of hours studied.

The goal is to help an EdTech platform identify students who may need additional academic support before final examinations.

---

## 🎯 Objective
- Predict student marks using study hours
- Use numerical data only
- Apply a supervised Machine Learning regression model

---

## 🧠 Machine Learning Technique Used
- **Algorithm:** Simple Linear Regression  
- **Learning Type:** Supervised Learning  
- **Problem Type:** Regression

---

## 📁 Dataset
The dataset consists of two columns:

| Column Name | Description |
|------------|-------------|
| Hours | Number of hours studied |
| Score | Student exam score |

Example:

| Hours | Score |
|-----------|-------|
| 1 | 45 |
| 2 | 50 |
| 3 | 60 |
| 4 | 70 |
| 5 | 78 |

---

## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## ⚙️ How the Model Works
The model learns a linear relationship between study hours and scores using the equation:

Score = m × StudyHours + c

Where:
- **m** is the slope
- **c** is the intercept
